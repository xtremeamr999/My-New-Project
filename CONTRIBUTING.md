# How to Contribute

In the pull request description, label what you did in the pr with feature, change, or bug fix.
Pull requests should target modern-dev, which is used for new features, changes, etcetera, before they are stable and can be merged into modern. All new code should be written in Java if possible; there are no plans to use Kotlin for the near future.

If you are unsure of what to do, please see SkyHanni's contributing guide https://github.com/hannibal002/SkyHanni/blob/beta/CONTRIBUTING.md, or if you have a more specific question, you can ask in the Discord server.
